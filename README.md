# Ascended Emotes!
### Extended message and emotes for WoW 3.3.5 Clients
Viva La Turtle 🐢

An addon that allows you to send messages in any channel, including emotes, that extend past the 255 character limit.

---

This was originally created for use in both ascension games- Ascension Bronzebeard and Epoch. It should be compatible with any 3.3.5 builds/servers.

This was also designed to be compatible with ElvUI and TotalRP3, and should be compatible with other UI overhauls as well. Having TotalRP3 installed also means that emotes that go past 1 message will have your name removed in the later messages.
